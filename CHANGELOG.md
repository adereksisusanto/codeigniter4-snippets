## [Released - 0.0.2]

- Add Snippets

| Command           | Description               | Output                                       |
| ----------------- | ------------------------- | -------------------------------------------- |
| ci4_extend        | Make <b>extend</b>        | `<?= $this->extend('code_here') ;?> `        |
| ci4_include       | Make <b>include</b>       | `<?= $this->include('code_here') ;?> `       |
| ci4_rendersection | Make <b>renderSection</b> | `<?= $this->renderSection('code_here') ;?> ` |
